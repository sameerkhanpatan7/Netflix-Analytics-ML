# 🎬 Netflix Analytics & Recommendation System

A Machine Learning based Netflix Recommendation System with an Interactive Analytics Dashboard built using Streamlit.

## Features

- Content-based Recommendation Engine
- Interactive Plotly Dashboard
- KPI Metrics
- Netflix Content Analytics
- Genre Analysis
- Country Analysis
- Ratings Distribution

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- TF-IDF Vectorization
- Cosine Similarity
- Plotly
- Streamlit

## Project Structure

```text
app/
model/
data/
notebook/
```

## Installation

```bash
git clone <repo-url>
cd Netflix-Analytics-Recommendation-System

pip install -r requirements.txt

streamlit run app/app.py
```
